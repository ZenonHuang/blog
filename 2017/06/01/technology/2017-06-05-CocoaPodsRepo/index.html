创建私有的 Spec Repo

在打造私有的组件之前，要首先建立一个私有的 Spec Repo.

Spec Repo 是所有的 Pods 的一个索引，就是一个容器，所有公开的 Pods 都在这个里面.实际只是一个Git仓库r.它的作用就像字典的目录一样，提供 Pods 的地址，版本等等

你可以选择 GitLab 或者 GitHub, Coding 等任何提供托管的地方，去建立一个自己的 Spec Repo。
